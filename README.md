# TODO APP RESTAPI
This App is a todolist Application with Nodejs/monogDB with a CRUD REST API for:
- Adding Tasks(Create).
- Deleting Task(Delete).
- Updating Task(Update).
- Show All Tasks(All).
## Requirements
- nodejs
- mongodb
## Running
```shell
git clone https://github.com/EngAhmedMahmoud/todoListAPI.git
cd todoListAPI
npm i
npm start
```
